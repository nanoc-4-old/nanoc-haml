# nanoc-haml

This provides a [Haml](http://haml.info/) filter for [nanoc](http://nanoc.ws).

## Installation

`gem install nanoc-haml`

## Usage

```ruby
filter :haml
```

Parameters passed to this filter will be passed on to Haml.
